## Set up

```
npm i
npm start
```

## Set up Troubleshooting

```
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.35.2/install.sh | bash
nvm install
nvm use
```

## Deploy

```
npm run deploy
```
